# 🎨 Optical Illusion Generator (Python)

A collection of black & white optical illusions generated using Python.

## ✨ Features
- Moving Squares Illusion
- Café Wall Illusion
- Rotating Snake Illusion
- High-resolution export (300 DPI)

## 🛠 Tech Stack
- Python
- NumPy
- Matplotlib

## 🚀 How to Run
```bash
pip install -r requirements.txt
python main.py
